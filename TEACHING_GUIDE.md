# Teaching Guide

## 운영 원칙

학생에게 처음부터 코드를 많이 쓰게 하지 않습니다. 먼저 `concept.md`를 읽고, `lesson.py`의 주석을 따라가며, 각 줄이 만드는 값을 말로 설명하게 한다.

## 90분 수업 흐름

1. 15분: 오늘 개념의 위치와 읽을 용어 설명
2. 25분: `lesson.py`를 실행하지 않고 주석과 코드만 읽기
3. 20분: Colab에서 실행하고 예측과 출력 비교
4. 20분: 기준값이나 입력값 하나만 바꾸어 출력 변화 관찰
5. 10분: 오늘의 핵심 문장을 학생 말로 다시 설명

## 질문 방식

- 이 줄은 어떤 값을 만드는가?
- 이 값의 타입은 무엇인가?
- 이 변수는 어디에서 처음 만들어졌는가?
- 이 출력은 어떤 계산에서 나왔는가?
- 실제 환자 데이터라면 어떤 개인정보 문제가 생길 수 있는가?
